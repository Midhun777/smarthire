Project Abstract: Job AI

Job AI is a modern recruitment platform that leverages Artificial Intelligence to match job seekers with relevant opportunities. The system parses resumes, extracts key skills, and provides a match score based on job requirements, streamlining the application process for both candidates and employers.

1. Database Schema

User Table

Field Type Description
_idObjectIdUnique identifier
nameStringFull name of the user
emailStringUnique email address
roleStringuser, admin, job_seeker, job_provider
skillsArray[String]Extracted skills
experienceArray[Object]Work experience details
isProfileCompleteBooleanProfile completion status

Job Table

Field Type Description
_idObjectIdUnique identifier
titleStringJob title
companyStringHiring company
locationStringJob location
requirementsArray[String]Key job requirements
salaryRangeStringExpected salary
postedByObjectIdReference to User (job_provider)

Application Table

Field Type Description
_idObjectIdUnique identifier
jobObjectIdReference to Job
userObjectIdReference to User (job_seeker)
statusStringPending, Reviewed, Accepted, etc.
appliedAtDateTimestamp of application
